# Projeto - Backend Flask (versão enxuta)

## Arquivos principais
- `app.py` - entrada do aplicativo e rotas.
- `models.py` - modelos SQLAlchemy (Usuario, Atleta, Video).
- `requirements.txt` - dependências.

## Instalação
1. Criar virtualenv e ativar:
   ```bash
   python -m venv .venv
   source .venv/bin/activate   # linux/mac
   .venv\Scripts\activate      # windows

## Execução

2. Para executar o programa, digite "python -m uvicorn main:app --reload" no terminal dentro da pasta "back".
